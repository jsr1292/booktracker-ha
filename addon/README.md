# Book Tracker Home Assistant Addon

Personal book tracking library with barcode scanning and reading statistics.

## Installation (Local Addon)

1. Copy this `addon` folder to your HA's `/addons/local/book_tracker/`
2. Restart Home Assistant
3. Go to Settings → Add-ons → Add-on Store → Local add-ons
4. Find "Book Tracker" and click Install

## Features
- Barcode scanning for adding books
- Reading statistics and timeline
- Genre and rating tracking
- Offline-capable PWA
